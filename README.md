# 資産ポートフォリオシミュレータ

6資産クラスを対象に、制約付きのモンテカルロ法で効率的フロンティアを近似するツールです。

対象資産:

- 日本株式
- 日本債券
- 先進国株式
- 先進国債券
- 新興国株式
- 新興国債券

## できること

- リスク・リターン散布図の生成
- 効率的フロンティアの描画
- 許容リスクごとに最大期待リターンとなる資産配分の CSV 出力

## Web 版の使い方

[index.html](/Users/yskmjp/Documents/codex/portfolio-simulator/web/index.html) をブラウザで開いて使えます。

実データを使う流れ:

1. 各資産の `公式データを開く` を押す
2. 三菱UFJアセットマネジメント公式の `chart_data_<fund_code>.js` を開く
3. ブラウザの `ファイル` → `名前を付けて保存` で、そのまま `.js` ファイルとして保存する
4. 画面に戻って6ファイルを選択する
5. `ファイル読込` を押す
6. `シミュレーション実行` を押す

補足:

- Yahoo!ファイナンス保存 HTML は件数が少なく、相関計算用には期間不足になりやすいです
- eMAXIS Slim の既存4資産は公式の `chart_data_*.js` を推奨します
- 新興国株式・新興国債券は、取得できる公式 CSV または保存HTMLも読み込めます
- どの頻度で計算したかは、画面上の「対象期間」と「注記」に表示されます

画面でできること:

- 各資産の最小配分・最大配分を入力
- 許容最大リスク、シミュレーション回数、刻み数、乱数シードを入力
- 公式の `chart_data_*.js` を読み込み、相関係数、期待リターン、ボラティリティを再計算
- 保存ファイルの期間が短い場合は、日次リターンの短期データから年率換算で推計
- 効率的フロンティアのグラフ表示
- 許容リスクごとの最適配分テーブル表示
- 相関行列の表示
- CSV ダウンロード

採用しているファンド:

- 日本株式: `eMAXIS Slim 国内株式（TOPIX）`
- 日本債券: `eMAXIS Slim 国内債券インデックス`
- 先進国株式: `eMAXIS Slim 先進国株式インデックス`
- 先進国債券: `eMAXIS Slim 先進国債券インデックス`
- 新興国株式: `eMAXIS Slim 新興国株式インデックス`
- 新興国債券: `iFree 新興国債券インデックス`

注記:

- サーバーは不要です
- 実データはブラウザ内だけで処理し、永続保存しません
- eMAXIS Slim そのものの時系列ページを使います

## Python 版の実行方法

```bash
python3 scripts/portfolio_simulator.py \
  --bounds '日本株式:0.1:0.5,日本債券:0.0:0.6,先進国株式:0.1:0.6,先進国債券:0.0:0.5' \
  --max-risk 0.18 \
  --simulations 30000 \
  --steps 10
```

## 入力

- `--bounds`: 各資産の最小配分・最大配分
- `--max-risk`: 許容できる最大リスク
- `--simulations`: シミュレーション回数
- `--steps`: 許容リスクの評価点数

## 出力

Python 版では `output/` 配下に次のファイルを生成します。

- `efficient_frontier.csv`: 許容リスクごとの最適配分
- `efficient_frontier.svg`: リスク・リターングラフ
- `summary.txt`: 人が読みやすい要約

## 前提

- 期待リターン、ボラティリティ、相関係数は `portfolio_simulator.py` 内で仮定値を設定しています
- 実際の市場データに合わせたい場合は `ASSETS` と `CORRELATION` を更新してください
